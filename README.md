## requirements
- python@3.12
- google-cloud-sdk
- make

```bash
sudo apt update
sudo apt install make google-cloud-sdk

curl -fsSL https://pyenv.run | bash
pyenv install 3.12
```

## installation
```bash
make install
```

## run 
```bash
make run
```

## evaluate
```bash
make evaluate
```

## summarize results
```bash
make summarize-reults
```
